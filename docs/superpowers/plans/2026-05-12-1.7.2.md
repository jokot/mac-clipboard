# MaClip 1.7.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship MaClip 1.7.2 covering five items: (1) restore Warp screenshot capture by diagnosing then dropping the over-broad concealed UTI; (2) extend the concealed auto-clear picker with 5s/10s/20s plus a "Never" entry; (3) auto-dismiss the overlay after idle (default ON, 20s); (4) reveal-masked-content eye button on concealed rows; (5) ESC + opening-shortcut close for Settings and Info windows.

**Architecture:** B-Warp uses a temporary `NSLog` in `ClipboardMonitor` to identify the offending UTI, then a one-line fix removes it from the concealed-UTI set. F1+F3 extends the picker entries and reserves `-1` as a "Never" sentinel honored by an `expiry == Date.distantFuture` rule in both capture and sweep paths. F7 idle-dismiss is implemented via a 1 s `Timer` plus an `NSEvent.addLocalMonitorForEvents` activity tracker on `OverlayWindowController`. F2 reveal uses a hover-visible `Button` with `@State` per row controlling a 5 s reveal window. ESC/shortcut additions extend the existing `SettingsWindowController` local monitor and add the equivalent to the Info window controller.

**Tech Stack:** Swift, SwiftUI, AppKit, Combine, XCTest. Module: `MaClip`. Build: `xcodebuild -scheme MacClipboard -destination 'platform=macOS' test`.

**Reference spec:** `docs/superpowers/specs/2026-05-12-1.7.2-design.md`

- [ ] **Step 4: Add idle tracking to `OverlayWindowController`**

Edit `App/OverlayWindow.swift`. Add properties to `final class OverlayWindowController`:

```swift
private var idleTimer: Timer?
private var idleMonitor: Any?
private var lastActivity: Date = Date()
```

Add helpers (place near the existing `cleanup` / `hide…` methods):

```swift
private func startIdleTracking() {
    stopIdleTracking()
    guard AppSettings.shared.autoDismissOverlayEnabled else { return }
    lastActivity = Date()
    idleMonitor = NSEvent.addLocalMonitorForEvents(
        matching: [.keyDown, .keyUp, .mouseMoved, .leftMouseDown, .rightMouseDown, .scrollWheel]
    ) { [weak self] event in
        self?.lastActivity = Date()
        return event
    }
    idleTimer = Timer.scheduledTimer(withTimeInterval: 1.0, repeats: true) { [weak self] _ in
        Task { @MainActor in
            guard let self else { return }
            let elapsed = Date().timeIntervalSince(self.lastActivity)
            if elapsed >= AppSettings.shared.autoDismissOverlayTimeout {
                self.stopIdleTracking()
                self.hideImmediatelyRefocusOnly()
            }
        }
    }
}

private func stopIdleTracking() {
    idleTimer?.invalidate()
    idleTimer = nil
    if let idleMonitor {
        NSEvent.removeMonitor(idleMonitor)
        self.idleMonitor = nil
    }
}
```

In `show()`, add `startIdleTracking()` at the very end of the method.

In every method that hides the overlay (`hide()`, `hideImmediatelyAndPaste()`, `hideImmediatelyRefocusOnly()`), call `stopIdleTracking()` at the start.

```swift
NSEvent.addLocalMonitorForEvents(matching: .keyDown) { [weak self] event in
    guard let self = self, self.window?.isKeyWindow == true else { return event }
    if event.keyCode == 53 {   // Escape
        self.window?.close()
        return nil
    }
    if event.modifierFlags.contains(.command), event.charactersIgnoringModifiers == "," {
        self.window?.close()
        return nil
    }
    return event
}
```
